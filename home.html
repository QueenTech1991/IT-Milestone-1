Welcome to IT 5443 Web Development Milestone #1 HTML/CSS

Course Description


This course introduces World Wide Web as a fundamental application platform for today's information systems. Students will examine core aspects of web technologies, web applications and will develop secured usable websites.
This course is part of MSIT foundation courses that aims to provide a quick coverage of fundamental IT knowledge that is needed for later MSIT studies.Ts.


Course Objectives


    Explain different components of World Wide Web as a platform.
  1. Design and develop websites using fundamental web languages, technologies, and tools.
  2. Distinguish between server-side and client-side web technologies
  3. Analyze user experienc, usability and accessibility issues related to web sites.
  4. Evaluate various web application development issues and trends.
  5. Conduct independent research on a subject related to the course material
